THE TALKING DOOR

A branding system for an outdoor tea party restaurant. Each component engages the viewer to interact and play with it reinforcing the restaurant's whimsical theme. Design includes a picture frame table top menu, a box containing a five foot long die-cute pull out menu, packaging for tea, and a website featuring a matching outdoor theme.
